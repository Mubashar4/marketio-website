Security

Security at Marketio.

We take the security of your data seriously. Here’s how we protect it.

Infrastructure Security

Marketio is hosted entirely on Amazon Web Services (AWS). Our infrastructure uses network segmentation, access controls, and continuous monitoring. We operate across multiple availability zones for high availability.

Data Encryption

All data transmitted to and from Marketio is encrypted in transit using TLS 1.2 or higher. Data stored on our systems is encrypted at rest using AES-256 encryption.

Access Controls

Access to production systems is restricted to authorised personnel only, with multi-factor authentication (MFA) enforced across all internal accounts. We operate on a principle of least privilege.

Application Security

We follow secure development practices and conduct regular security reviews. Our AI components, powered by Anthropic Claude, are deployed within access-controlled environments and are not permitted to exfiltrate customer data.

Incident Response

In the event of a security incident affecting your data, Marketio will notify affected customers within 72 hours of becoming aware of the breach (or as required by applicable law).

Compliance Alignment

Marketio is designed with compliance in mind, including alignment with GDPR, CCPA, CAN-SPAM, and CASL. For Data Processing Agreements (DPAs), contact support@marketio.net.

Vulnerability Disclosure

Found a security issue?

If you’ve discovered a potential vulnerability in our platform, please disclose it responsibly. We commit to acknowledging receipt within 48 hours and not pursuing legal action against good-faith security researchers.
